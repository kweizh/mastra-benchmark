## What was built

### `src/index.ts`
- **`InMemoryStore`** – shared storage backend for all agents.
- **`sharedMemory`** – a single `Memory` instance (from `@mastra/memory`) wired to the store; passed to all three agents so conversational context persists across delegations.
- **`coder`** agent – `openai/gpt-4o-mini`, instructions: write clean TypeScript.
- **`reviewer`** agent – `openai/gpt-4o-mini`, instructions: security + performance review.
- **Two delegation tools** (`delegate-to-coder`, `delegate-to-reviewer`) created with `createTool`; each calls the respective agent's `generate()` with `memory: { thread, resource }` so every sub-call shares the same thread.
- **`supervisor`** agent – orchestrates coder → reviewer, then synthesises both outputs; tools include the two delegation tools.
- **`mastra`** instance – registers all three agents (`coder`, `reviewer`, `supervisor`) and the storage.

### `src/run.ts`
1. Resolves the supervisor agent from the Mastra instance.
2. Sends `"Write a simple Express server and review it"` with a shared `threadId` / `resourceId`.
3. Prints the response to stdout and writes it to `/home/user/project/output.txt`.
